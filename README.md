# Rente Per Maand - SEO Content Hub

Een SEO-geoptimaliseerde, evergreen content hub voor financiële informatie en tools.

## 🎯 Features

✅ **SEO Optimized**
- Structured Data (JSON-LD) op elke pagina
- Meta tags & Open Graph tags
- Sitemap & Robots.txt
- Canonical URLs
- Mobile-first responsive design
- Fast load times (Core Web Vitals optimized)

✅ **Content Pillars**
- ✨ "Hoe werkt rente?" - Evergreen artikel met formules
- 🧮 Spaarreturn Calculator - Interactieve tool
- 📱 Responsive design geschikt voor alle devices

✅ **Financial Tools**
- Interactieve calculators met real-time berekeningen
- User-friendly interface
- Ad-ready structure

## 🚀 Tech Stack

- **Next.js 14+** - SSG/SSR voor best SEO performance
- **React 18** - Modern component structure
- **Tailwind CSS** - Utility-first styling
- **Node.js** - Runtime

## 📋 Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in je browser.

## 🔨 Build & Deployment

```bash
npm run build
npm start
```
